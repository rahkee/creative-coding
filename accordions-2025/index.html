Advanced Accordions 2025

Using the new HTML name attribute for exclusive accordion behavior

What is the new accordion feature?

The new HTML name attribute allows you to group <details> elements together, ensuring only one accordion is open at a time - all without JavaScript!

How does it work?

By adding the same name attribute to multiple <details> elements, the browser automatically handles the exclusive behavior. When you open one accordion, any other accordion with the same name will automatically close.

This is a native browser feature that works seamlessly across modern browsers, providing better performance and accessibility out of the box.

What are the benefits of using native accordions?

Native Browser Support

The <details> element is built into the browser, which means it works without any JavaScript libraries or frameworks. This results in faster load times and better performance.

Accessibility

Native accordions come with built-in accessibility features. Screen readers automatically understand the expand/collapse behavior, and keyboard navigation works out of the box with proper focus management.

Progressive Enhancement

Even if CSS fails to load or JavaScript is disabled, the accordion still functions perfectly. Users can still expand and collapse content, ensuring a resilient user experience.

Maintainability

Using native HTML elements means less code to maintain, fewer bugs, and better long-term stability. You don't need to worry about updating third-party libraries or dealing with breaking changes.

Performance

Browser-native features are optimized at the engine level, providing smoother animations and better resource management compared to JavaScript-based solutions.

Browser Support

The name attribute for exclusive accordions is supported in:

  • Chrome 120+
  • Edge 120+
  • Safari 17.2+
  • Firefox 120+
Code Example

Here's how simple it is to create grouped accordions:

<details name="accordion">
  <summary>First Item</summary>
  <div class="accordion-content">
    <p>Content here</p>
  </div>
</details>

<details name="accordion">
  <summary>Second Item</summary>
  <div class="accordion-content">
    <p>More content</p>
  </div>
</details>

That's all you need! The browser handles the rest.
